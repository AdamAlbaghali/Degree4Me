About Computer Science

The Computer Science degree program equips students with theoretical knowledge and practical skills in software development, algorithms, data structures, and computer systems. It prepares students for careers in software engineering, data science, cybersecurity, and more.

Core Courses

  • Introduction to Computer Science
  • Data Structures and Algorithms
  • Programming Languages
  • Computer Architecture
  • Operating Systems
  • Database Management Systems
  • Software Engineering
  • Computer Networks
  • Artificial Intelligence
  • Security and Cryptography
